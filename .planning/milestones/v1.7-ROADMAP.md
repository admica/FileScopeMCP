# Milestone v1.7: Multi-Lang Symbols + Call-Site Edges

**Status:** ✅ SHIPPED 2026-04-24
**Phases:** 36-39
**Total Plans:** 8

## Overview

Extended symbol-level intelligence to Python/Go/Ruby via tree-sitter grammars, upgraded TS/JS dependency edges from file-granular to call-site-granular with `symbol_dependencies` table, shipped `find_callers`/`find_callees` MCP tools so agents can answer "who calls foo", and formally closed all 7 historical deferred quick-task items.

**Plans**: 3 plans

Plans:
- [x] 36-01: Perf baseline (`v1.7-baseline.json`) + `symbol_dependencies` Drizzle schema + SQL migration 0006 + `SymbolKind` += `'module'` / `'struct'` + `npm install tree-sitter-go@0.25.0 tree-sitter-ruby@0.23.1`
- [x] 36-02: Inline Python/Go/Ruby symbol extractors in `src/language-config.ts` + `extractLangFileParse()` export + three-way coordinator pass-2 dispatch + `find_symbol` description update
- [x] 36-03: `src/migrate/bulk-multilang-symbol-extract.ts` (three independent sub-passes, per-language `kv_state` gates) + coordinator startup wiring + permanent single-pass-invariant test + `36-VERIFICATION.md`

**Details:**

**Requirements**: CSE-02, CSE-03, CSE-04, CSE-05, CSE-06
**Plans**: 2 plans

Plans:

- [x] 37-02: Five-step `deleteFile()` cascade; `watcher-symbol-lifecycle.test.ts` regression; `bulk-call-site-extract.ts` gated on three Phase 36 precondition keys; coordinator wiring; `37-VERIFICATION.md`
