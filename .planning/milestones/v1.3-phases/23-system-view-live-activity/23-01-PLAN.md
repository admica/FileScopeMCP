**2. Create `src/nexus/log-tailer.ts`** with these exports:

```typescript
export type LogLine = {
  timestamp: string;   // ISO timestamp extracted from log line
  prefix: string;      // e.g. "INFO", "WORKER" — empty string if none
  message: string;     // remainder after timestamp and optional prefix
  source: 'broker' | 'mcp-server';
};
```

**Module constants:**
- `RING_BUFFER_SIZE = 500`
- `LOG_PATHS`: array of two paths: `path.join(FILESCOPE_DIR, 'broker.log')` and `path.join(FILESCOPE_DIR, 'mcp-server.log')`. Import `FILESCOPE_DIR` from `../broker/config.js`.

**Module state:**
- `ringBuffer: LogLine[]` — array, max 500 entries, `shift()` when full before `push()`
- `sseClients: Set<http.ServerResponse>` — import `http` from `node:http`
- `fileStates: Map<string, { offset: number; watcher: fs.FSWatcher | null }>` — per-file tracking

**Parse regex (NEXUS-30):**
```typescript
const LOG_REGEX = /^\[([^\]]+)\]\s+(?:\[([^\]]+)\]\s+)?(.*)$/;
```
This regex handles both formats:
- `[2026-04-01T02:32:33.150Z] BrokerWorker started` -> timestamp=ISO, prefix='', message='BrokerWorker started'
- `[2026-04-01T02:32:32.500Z] [INFO] Using default config` -> timestamp=ISO, prefix='INFO', message='Using default config'

**Exported functions:**

`getRecentLines(): LogLine[]` — returns copy of ringBuffer via `[...ringBuffer]`

`addSseClient(res: http.ServerResponse): () => void` — adds res to sseClients Set, returns cleanup function that deletes from Set

`initLogTailer(): void` — for each path in LOG_PATHS:
  1. Determine `source` from path: `'broker'` if path includes 'broker', else `'mcp-server'`
  2. Create FileState `{ offset: 0, watcher: null }`
  3. If file exists (`fs.existsSync`): read entire file, split by `\n`, take last 500 lines, parse each with `parseLine()`, append valid lines to ringBuffer. Set `state.offset = fs.statSync(logPath).size`.
  4. Try `fs.watch(logPath, { persistent: false }, callback)` — on callback, call `readNewBytes(logPath, state, source)`.
  5. If `fs.watch` throws (ENOENT — file doesn't exist yet), set `watcher = null` and skip. The file will be picked up if it appears later (the periodic 5s poll from the broker status polling effectively covers this since it triggers UI refresh).
  6. Store state in `fileStates` Map.

`stopLogTailer(): void` — iterate fileStates, close each watcher, clear the Map, clear sseClients Set (write a final empty line or just delete — clients will get socket close event).

**Internal functions:**

`parseLine(raw: string, source: LogLine['source']): LogLine | null` — trim, return null if empty, match against LOG_REGEX, return null if no match, otherwise return `{ timestamp: m[1], prefix: m[2] ?? '', message: m[3], source }`.

`appendToBuffer(line: LogLine): void` — if `ringBuffer.length >= RING_BUFFER_SIZE`, `ringBuffer.shift()`. Then `ringBuffer.push(line)`.

`broadcast(line: LogLine): void` — format as `data: ${JSON.stringify(line)}\n\n`, iterate sseClients, try `client.write(payload)`, catch errors and delete client from Set.

`readNewBytes(filePath: string, state: FileState, source: LogLine['source']): void`:
  1. `const stat = fs.statSync(filePath)` — wrap in try/catch, return on error
  2. Log rotation: if `stat.size < state.offset`, reset `state.offset = 0`
  3. If `stat.size <= state.offset`, return (no new data)
  4. Allocate Buffer of `stat.size - state.offset`
  5. `fs.openSync(filePath, 'r')`, `fs.readSync(fd, buf, 0, buf.length, state.offset)`, `fs.closeSync(fd)`
  6. Update `state.offset += bytesRead`
  7. Split buffer text by `\n`, parse each line, for valid lines call `appendToBuffer(line)` then `broadcast(line)`

**Add `queryBrokerStatus()` function** inside server.ts (private to the module, before routes):
```typescript
async function queryBrokerStatus(): Promise<StatusResponse | null> {
  return new Promise<StatusResponse | null>((resolve) => {
    const sock = net.createConnection(SOCK_PATH);
    const timer = setTimeout(() => { sock.destroy(); resolve(null); }, 2000);
    timer.unref();

    sock.on('error', () => { clearTimeout(timer); resolve(null); });

    const rl = readline.createInterface({ input: sock, terminal: false });
    rl.on('line', (line) => {
      try {
        const msg = JSON.parse(line);
        if (msg.type === 'status_response') {
          clearTimeout(timer);
          sock.destroy();
          resolve(msg as StatusResponse);
        }
      } catch { /* ignore malformed */ }
    });

    const id = randomUUID();
    sock.on('connect', () => {
      sock.write(JSON.stringify({ type: 'status', id }) + '\n');
    });
  });
}
```

app.get('/api/system/tokens', async (_req, _reply) => {
  const current = readStats();
  const snapshot = options.startupTokenSnapshot;
  const entries = Object.entries(current.repoTokens)
    .map(([repo, total]) => ({
      repo,
      total,
      sessionDelta: total - (snapshot[repo] ?? total),

});

Route 3 — `GET /api/stream/activity` SSE endpoint (NEXUS-28, NEXUS-29):
```typescript
app.get('/api/stream/activity', async (req, reply) => {
  reply.raw.setHeader('Content-Type', 'text/event-stream');
  reply.raw.setHeader('Cache-Control', 'no-cache');
  reply.raw.setHeader('Connection', 'keep-alive');
  reply.raw.setHeader('X-Accel-Buffering', 'no');
  reply.raw.flushHeaders();

  // Send ring buffer history immediately
  for (const line of getRecentLines()) {
    reply.raw.write(`data: ${JSON.stringify(line)}\n\n`);
  }

  // Register for live broadcast
  const cleanup = addSseClient(reply.raw);
  req.socket.on('close', cleanup);

  // Prevent Fastify from trying to serialize a response
  await reply.hijack();
});
```
