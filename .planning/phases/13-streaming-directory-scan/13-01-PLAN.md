<objective>
Convert `scanDirectory` from an eager tree-building function to an async generator that yields one FileNode at a time using `fs.promises.opendir`, and update all tests to consume the generator via a `collectStream()` helper.

Purpose: This is the first half of PERF-02. The generator replaces `readdir` with `opendir` (no directory listing buffering) and yields files lazily instead of building a full nested tree in memory. The coordinator integration (Pass 1 streaming to SQLite, Pass 2 dependency resolution) is handled in Plan 02.

Output: `scanDirectory` returns `AsyncGenerator<FileNode>` instead of `Promise<FileNode>`. All tests pass using `collectStream()`.
</objective>
