```typescript
export async function connect(repo: string): Promise<void>

```

**Module-level state variables:**
```typescript
let socket: net.Socket | null = null;
let reconnectTimer: ReturnType<typeof setInterval> | null = null;
let repoPath: string = '';
let _intentionalDisconnect: boolean = false;
```

**Imports (all from Node.js built-ins or existing project modules):**
```typescript
import * as net from 'node:net';
import * as readline from 'node:readline';
import { randomUUID } from 'node:crypto';
import { existsSync, readFileSync } from 'node:fs';
import { spawn } from 'node:child_process';
import { fileURLToPath } from 'node:url';
import * as path from 'node:path';
import { SOCK_PATH } from './config.js';
import type { SubmitMessage, BrokerMessage } from './types.js';
import { writeLlmResult, clearStaleness } from '../db/repository.js';
import { getSqlite } from '../db/db.js';
import { log } from '../logger.js';
```

**`isConnected()` function:**
Returns `socket !== null && !socket.destroyed`.

**`connect(repo)` function (async):**
1. Set `repoPath = repo` and `_intentionalDisconnect = false`
2. Call `spawnBrokerIfNeeded()` — if `!existsSync(SOCK_PATH)`, spawn the broker binary:
   - Resolve broker binary path: `const distBrokerDir = path.dirname(fileURLToPath(import.meta.url));` then `const brokerBin = path.resolve(distBrokerDir, 'main.js');`
   - `spawn(process.execPath, [brokerBin], { detached: true, stdio: 'ignore' }).unref()`
   - Wrap in try/catch — log warning on failure, do NOT throw
   - After spawn, wait 500ms (`await new Promise(r => setTimeout(r, 500))`) to let broker bind socket
3. Call `await attemptConnect()` — wraps the actual socket connection in a Promise

**`attemptConnect()` function (private, returns Promise<void>):**
1. If socket is already connected and not destroyed, return immediately
2. Create socket: `const sock = net.createConnection(SOCK_PATH)`
3. On 'connect' event:
   - Set `socket = sock`
   - Clear reconnect timer
   - Attach NDJSON readline listener via `attachListener(sock)`
   - Log `[broker-client] Connected to broker`
   - Call `resubmitStaleFiles()` to send all stale files
   - Resolve the promise
4. On 'error' event:
   - Log `[broker-client] Connection error: ${err.message}`
   - Start reconnect timer
   - Resolve the promise (do NOT reject — connect is best-effort)
5. On 'close' event:
   - Set `socket = null`
   - If NOT `_intentionalDisconnect`, start reconnect timer and log `[broker-client] Disconnected — reconnecting in 10s`

**`disconnect()` function:**
1. Set `_intentionalDisconnect = true`
2. Call `clearReconnectTimer()`
3. If socket exists: `socket.destroy(); socket = null`

**`submitJob(filePath, jobType, importance, fileContent, payload?)` function:**
1. If `!isConnected()` return immediately (silent drop)
2. Build a `SubmitMessage` object: `{ type: 'submit', id: randomUUID(), repoPath, filePath, jobType, importance, fileContent, payload }`
3. Write to socket: `socket!.write(JSON.stringify(msg) + '\n')`

**`attachListener(sock)` function (private):**
1. Create readline interface: `readline.createInterface({ input: sock, terminal: false })`
2. On 'line': parse JSON as `BrokerMessage`, call `handleBrokerMessage(msg)`
3. Wrap JSON.parse in try/catch — log malformed messages

**`handleBrokerMessage(msg)` function (private):**
1. If `msg.type === 'result'`: call `writeLlmResult(msg.filePath, msg.jobType, msg.text)`, then `clearStaleness(msg.filePath, msg.jobType)`, then log the result
2. If `msg.type === 'error'`: log it with `msg.code` and `msg.message`. No other action — file stays stale for resubmission.

**`resubmitStaleFiles()` function (private):**
1. Get SQLite handle: `const sqlite = getSqlite()`
2. Query stale files:
```sql
SELECT path, importance,
       summary_stale_since, concepts_stale_since, change_impact_stale_since
FROM files
WHERE summary_stale_since IS NOT NULL
   OR concepts_stale_since IS NOT NULL
   OR change_impact_stale_since IS NOT NULL
ORDER BY importance DESC
```
3. For each row, read file content with `readFileSync(row.path, 'utf-8')` wrapped in try/catch (skip on error — file may have been deleted)
4. For each non-null stale column, call `submitJob(row.path, jobType, row.importance, fileContent)`
5. Log count of submitted jobs

**`startReconnectTimer()` function (private):**
1. If `reconnectTimer !== null` return (already running)
2. `reconnectTimer = setInterval(() => { attemptConnect().catch(() => {}); }, 10_000)`
3. `reconnectTimer.unref()` — critical for not blocking process exit

**`clearReconnectTimer()` function (private):**
1. If `reconnectTimer !== null`: `clearInterval(reconnectTimer); reconnectTimer = null`
