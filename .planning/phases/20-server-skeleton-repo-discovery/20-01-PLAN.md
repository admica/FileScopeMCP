**Step 2: Create `src/nexus/discover.ts`**

This module handles the nexus.json registry and auto-discovery.

Types to export:
```typescript
export type NexusRepo = { path: string; name: string };
export type NexusRegistry = { repos: NexusRepo[] };
```

Constants:
```typescript
import { FILESCOPE_DIR } from '../broker/config.js';
export const NEXUS_JSON_PATH = path.join(FILESCOPE_DIR, 'nexus.json');
```

Functions to implement:

1. `readRegistry(): NexusRegistry | null` — Read and parse `~/.filescope/nexus.json`. Return null if file doesn't exist or is unparseable (try/catch around readFileSync + JSON.parse).

2. `writeRegistry(registry: NexusRegistry): void` — Write registry to `NEXUS_JSON_PATH` with `JSON.stringify(registry, null, 2)`. Create `FILESCOPE_DIR` with `fs.mkdirSync(FILESCOPE_DIR, { recursive: true })` before writing.

3. `async discoverRepos(): Promise<NexusRepo[]>` — Scan 2 levels deep from `os.homedir()` for `.filescope/data.db`:
   - Use `fs.promises.glob('*/.filescope/data.db', { cwd: home })` for level 1
   - Use `fs.promises.glob('*/*/.filescope/data.db', { cwd: home })` for level 2
   - For each match, extract the repo root by stripping `/.filescope/data.db` suffix using `path.dirname(path.dirname(match))` (since match is relative like `foo/.filescope/data.db`, dirname gives `foo/.filescope`, dirname again gives `foo`, then join with home)
   - Actually: match = `ProjectName/.filescope/data.db`. `path.dirname(match)` = `ProjectName/.filescope`. We want `ProjectName`. So: `path.join(home, match)` gives full path to data.db, then go up 2 dirs: `path.resolve(path.join(home, match), '../..')` to get the repo root.
   - Derive `name` from `path.basename(repoRoot)`.
   - Deduplicate by repo path using `new Set()`.
   - Return array of `{ path: repoRoot, name }`.

**Step 3: Create `src/nexus/repo-store.ts`**

This module manages per-repo read-only better-sqlite3 connections.

Use the EXACT `createRequire` pattern from `src/db/db.ts`:
```typescript
import { createRequire } from 'node:module';
const _require = createRequire(import.meta.url);
const Database = _require('better-sqlite3') as typeof import('better-sqlite3');
```

Types:
```typescript
export type RepoState = {
  name: string;
  path: string;
  db: InstanceType<typeof Database> | null;
  online: boolean;
};
```

Module state: `const repos = new Map<string, RepoState>();`

Functions to implement:

1. `openRepo(name: string, repoPath: string): RepoState` — Check if `.filescope/data.db` exists at `path.join(repoPath, '.filescope', 'data.db')` using `fs.existsSync()`. If missing, store as `{ name, path: repoPath, db: null, online: false }`. If exists, open with `new Database(dbPath, { readonly: true })`, set `db.pragma('cache_size = -32000')` (32MB cache). Do NOT set `journal_mode = WAL` (read-only connection cannot change journal mode — WAL already set by MCP writer). Store as `{ name, path: repoPath, db, online: true }`. Return the state.

2. `getRepos(): RepoState[]` — Return `Array.from(repos.values())`.

3. `getDb(repoName: string): InstanceType<typeof Database> | null` — Look up repo by name, return `state.db` (which is null if offline).

4. `closeAll(): void` — Iterate all repos, close any open db connections (`state.db.close()`), set `state.db = null`.

5. `recheckOffline(): void` — Iterate repos where `online === false`. For each, check if `data.db` now exists. If yes, open it (same as `openRepo` logic), update the entry in the map. Log `Nexus: repo ${name} came online` on reconnect.

6. `getRepoStats(db: InstanceType<typeof Database>): object` — Execute the stats query against a repo's db:
```sql
SELECT
  COUNT(*) FILTER (WHERE is_directory = 0) AS total_files,
  COUNT(*) FILTER (WHERE is_directory = 0 AND summary IS NOT NULL) AS with_summary,
  COUNT(*) FILTER (WHERE is_directory = 0 AND concepts IS NOT NULL) AS with_concepts,
  COUNT(*) FILTER (WHERE is_directory = 0 AND summary_stale_since IS NOT NULL) AS stale_count
FROM files
```
Plus a second query:
```sql
SELECT COUNT(*) AS total_deps FROM file_dependencies WHERE dependency_type = 'local_import'
```
Return `{ totalFiles, withSummary, withConcepts, staleCount, totalDeps }`.

All imports use `.js` extensions (ESM convention: `import { FILESCOPE_DIR } from '../broker/config.js'`).

**Step 2: Create `src/nexus/main.ts`**

This is the CLI entry point. First line MUST be the shebang comment:
```typescript
#!/usr/bin/env node
```

Implement `parseArgs()` function:
```typescript
function parseArgs(): { port: number; host: string } {
  const args = process.argv.slice(2);
  let port = 1234;
  let host = '0.0.0.0';
  for (let i = 0; i < args.length; i++) {
    if (args[i] === '--port' && args[i + 1]) port = parseInt(args[++i], 10);
    if (args[i] === '--host' && args[i + 1]) host = args[++i];
  }
  return { port, host };
}
```

Implement `async function main()`:

1. Parse CLI args via `parseArgs()`.

2. Discover repos:
```typescript
import { readRegistry, writeRegistry, discoverRepos } from './discover.js';
import { openRepo, closeAll, recheckOffline } from './repo-store.js';

let registry = readRegistry();
if (!registry) {
  console.log('Nexus: no nexus.json found, scanning for repos...');
  const discovered = await discoverRepos();
  registry = { repos: discovered };
  writeRegistry(registry);
  console.log(`Nexus: discovered ${discovered.length} repos, saved to nexus.json`);
}
```

3. Open all repo databases:
```typescript
for (const repo of registry.repos) {
  const state = openRepo(repo.name, repo.path);
  const status = state.online ? 'online' : 'OFFLINE';
  console.log(`  - ${repo.name} (${repo.path}) [${status}]`);
}
```

4. Start periodic recheck (60s interval):
```typescript
const recheckInterval = setInterval(() => recheckOffline(), 60_000);
recheckInterval.unref(); // don't keep process alive just for recheck
```

5. Create and start server:
```typescript
import { createServer } from './server.js';
import { fileURLToPath } from 'node:url';
import path from 'node:path';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const staticDir = path.join(__dirname, 'static');

const server = await createServer({ staticDir });
await server.listen({ port: config.port, host: config.host });
console.log(`Nexus: http://${config.host}:${config.port}`);
```
(Where `config` is the result of `parseArgs()`.)

6. Implement graceful shutdown:
```typescript
async function shutdown(signal: string) {
  console.log(`Nexus: received ${signal}, shutting down...`);
  const timeout = setTimeout(() => process.exit(1), 10_000);
  timeout.unref();
  clearInterval(recheckInterval);
  closeAll();
  await server.close();
  clearTimeout(timeout);
  process.exit(0);
}

process.on('SIGTERM', () => shutdown('SIGTERM'));
process.on('SIGINT', () => shutdown('SIGINT'));
```

7. Call `main()` at module level:
```typescript
main().catch((err) => {
  console.error('Nexus: fatal error', err);
  process.exit(1);
});
```
