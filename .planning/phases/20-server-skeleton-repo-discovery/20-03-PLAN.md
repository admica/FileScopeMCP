**Step 1: Build the existing project first (prerequisite for cross-directory imports)**

```bash
npm run build
```

This ensures `dist/broker/config.js` exists, which `dist/nexus/discover.js` imports via `../broker/config.js`.

**Step 2: Build the Nexus backend**

```bash
npm run build:nexus-api
```

Verify output:
- `dist/nexus/main.js` exists
- `dist/nexus/server.js` exists
- `dist/nexus/repo-store.js` exists
- `dist/nexus/discover.js` exists

**Step 3: Check shebang preservation**

```bash
head -1 dist/nexus/main.js
```

If the first line is NOT `#!/usr/bin/env node`, prepend it:
```bash
echo '#!/usr/bin/env node' | cat - dist/nexus/main.js > /tmp/nexus-main.js && mv /tmp/nexus-main.js dist/nexus/main.js
```

Add a postbuild script to package.json so this happens automatically. Check esbuild output first:
- If esbuild preserves the shebang comment: only add `chmod +x`
- If esbuild strips it: add prepend + chmod

The script to add (handle both cases):
```json
"postbuild:nexus-api": "grep -q '#!/usr/bin/env node' dist/nexus/main.js || (printf '#!/usr/bin/env node\\n' | cat - dist/nexus/main.js > /tmp/nexus-main.js && mv /tmp/nexus-main.js dist/nexus/main.js); chmod +x dist/nexus/main.js"
```

**Step 4: Build the Nexus frontend**

```bash
npm run build:nexus-ui
```

Verify output:
- `dist/nexus/static/index.html` exists
- `dist/nexus/static/assets/` contains `.js` and `.css` files

**Step 5: Verify cross-directory imports resolve at runtime**

```bash
grep 'broker/config' dist/nexus/discover.js
ls dist/broker/config.js
```

Both must succeed. If `dist/broker/config.js` is missing, Step 1 wasn't run.

**Step 6: Runtime smoke test**

```bash
node dist/nexus/main.js &
NEXUS_PID=$!
sleep 2
curl -s http://localhost:1234/api/repos | head -100
curl -s -o /dev/null -w "%{http_code}" http://localhost:1234/
kill $NEXUS_PID 2>/dev/null
wait $NEXUS_PID 2>/dev/null
```

If the smoke test reveals errors, fix them:
- Import path errors: fix the import in the source file, rebuild
- Missing module errors: run `npm run build` first
- Static serving errors: verify @fastify/static root path uses `fileURLToPath(new URL('.', import.meta.url))` + `'static'`
- Port conflicts: verify port 1234 is free

**Step 7: Add convenience build-all script**

```json
"build:nexus": "npm run build && npm run build:nexus-api && npm run build:nexus-ui"
```

This ensures the full project (including broker/config.js) is built before Nexus backend and frontend.
